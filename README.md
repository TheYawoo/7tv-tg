# 7tv-tg
Telegram Desktop Client with 7TV emotes. WIP.

# Dependencies
- Node.js v18 or newer

# Installation
1.  Download and unzip into some folder.
2.  Open cmd, go to your folder.
3.  `npm i electron`
4.  `npm i -D electron-builder cross-env`
5.  `npm run dev`
